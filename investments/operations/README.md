# Operations

This folder contains runtime outputs only.

## Allowed content

- Research result markdown files in `research_results/`
- Evidence and audit logs in `logs/`
- Review snapshots in `logs/snapshots/`

## Not allowed content

- Normative methodology and governance guides
- Canonical thesis logic

Use these paths as source of current state.
Use `../guides/active/investment_methodology.md` for process rules.
Use `../thesis/` for canonical strategy logic.
