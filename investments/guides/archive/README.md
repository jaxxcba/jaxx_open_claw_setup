# Archive Policy (Lean)

Status: Active
Last Updated: 2026-02-18

This archive follows a lean policy.

## Policy

- Keep one canonical file per meaningful version.
- Keep versioned files; remove duplicate imports and throwaway copies.
- Do not keep filename variants like ` (1)` when content is identical.
- Prefer version-stamped files over unversioned ad-hoc copies.
