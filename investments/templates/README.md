# Templates

This folder keeps only canonical templates with standardized names.

## Naming standard

Template files must follow:
- lowercase,
- snake_case or kebab-case,
- explicit version and date suffix,
- no spaces,
- no duplicate import suffixes like ` (1)`.

## Active templates

- `research-summary-template_v1.2.0_2026-02-10.md`
- `us_market_mini_brief_template_v1.0.0_2026-02-18.md`
